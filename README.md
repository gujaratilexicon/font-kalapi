How to generate TTF file?
=========================

In Linux (or any UNIX like systems or Mac OS X), run following in Terminal, you
need to have FontForge installed. See: <http://fontforge.sourceforge.net/>

 $mkdir TTFfiles

 $chmod +x generate.pe

 $./generate.pe KalapiUnicode.sfd
